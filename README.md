# ENTROP

Browser-local **entropy harvester** demo: **rAF** timing → **SHA-256** pool → optional **copy seed**. Canvas layers: fluid ASCII diffusion (`lib/fluid-smoke.ts`), variable serif glyph row (`measureNaturalWidth`), and **full-width pretext** body copy explaining entropy, honest limits, and a citizen-scale vision next to quantum/crypto concentration.

Not a replacement for OS CSPRNG or post-quantum products—an inspectable step toward **user-visible** randomness mixing.

```bash
npm install && npm run dev
```
